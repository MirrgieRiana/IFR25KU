# ミラージュフェアリー劇場ってなんなのだぁ🌱✨

「ミラージュフェアリー劇場」は、IFR25KUの公式サイトで公開されている、会話劇のかたちのWebコンテンツなのだぁ〜🌱♪
春日部つむぎとずんだもんの2人の掛け合いを通して、IFR25KUっていうMODの世界観・アイテム・文化を、物語として味わえる、連作の読み物なのだぁ✨

この文書は、劇場が「何であるか」を、まとめて考察したものなのだぁ〜🌱♪ ゲーム本体のポエムや分類やソースコードみたいな一次情報と、ボクの読み解きは、はっきり区別して書くのだぁ✨ ボクの読み解きの部分には、ちゃんとその旨を書いておくのだぁ🌱

## 劇場を網羅的に知りたいときの早見表なのだぁ🔍✨

この文書は、劇場の「考え方」や「作法」みたいな、変わりにくいところをまとめたものなのだぁ〜🌱♪ 個々の記事の中身や元ネタみたいに、記事が増えると育っていくところは、この文書には抱え込まないのだぁ✨ 抱え込むと、新しい記事が1本増えた瞬間に古びて、齟齬のもとになっちゃうからなのだぁ💧
劇場のことを、網羅的に・具体的に・正確に知りたくなったら、次の一次情報を見るのがいちばん確実なのだぁ🌱

| 知りたいこと | 見るべき一次情報 |

| つむぎ・ずんだもんのペルソナ | 同じ `skills/` 下の `kasukabe-tsumugi-persona`・`zundamon-persona` なのだぁ |

## 劇場のねらい — 文化を伝える読み物なのだぁ🌱

劇場の制作ルールは、`site/ifr25ku-site-claude-marketplace/ifr25ku-site/skills/theater-creator/SKILL.md` にちゃんと明文化されているのだぁ〜🌱♪
そこで宣言されている本当のねらいは、アップデート内容の報告やゲーム仕様の説明じゃなくて、「IFRKUという文化を多くの人に知ってもらうこと」なのだぁ✨
Web検索でのヒット → クリック → 読み始め → 釘付け → 別記事への誘導 → 再訪 → 共有、っていう読み手の行動の連鎖を狙う、集客戦略に立っているのだぁ🌱

逆に「入手方法・使い方・ゲーム仕様の説明」や「実際にゲームで遊びたいと思わせること」は、重要じゃないって割り切っているのだぁ〜🌱 Web上で会話形式の物語を読みたい人と、ゲームをしたい人は、別のことが多いからなのだぁ✨

このねらいを支える、ほぼ唯一の汎用戦略が「情報の空白」なのだぁ🌱♪ タイトルは `【主題】中身が気にならせるような会話の抜粋【つむぎ×ずんだもん】` の形をとって、タイトルを読んだだけじゃ解消できない疑問を残すことを大事にしているのだぁ✨
ただし、思わせぶりな言葉で露骨に謎をアピールするんじゃなくて、状況や事実を淡々と述べた結果として、自然に疑問が残るのが理想なのだぁ〜🌱 実際の記事タイトル「あーし石炭の火薬がいい！！」「溶岩の上でも関係ないもんね！」なんかは、キャラのセリフを淡々と抜き出しただけで、自然に「なんで？」が残る作りになっているのだぁ✨

## 劇場の心臓 — 「翻訳」なのだぁ🌱

制作ルールの中で、ボクがいちばん大事だと思っている考えが「翻訳」なのだぁ〜🌱♪ スキルには「ゲームと文学作品は異なる空気感で成立しており、それらを利用するには『翻訳』をしなければなりません」って書いてあるのだぁ✨

スキルで「翻訳」という言葉が実際に書かれているのは「ゲーム仕様の直接的な登場の禁止」のところなのだぁ🌱 でもボクは、そのすぐ隣の「ゲーム内テキストの使用の禁止」も、同じ考えで読めると思っているのだぁ✨ 2つを並べてみるのだぁ🌱

- **ゲーム仕様の直接的な登場の禁止**：レシピの格子みたいな、ゲーム側の都合をキャラが認識するのは不自然なのだぁ〜🌱 「棒の上に理天石が四つ」じゃなくて「棒の上に青いの」って、キャラの視点に翻訳するのだぁ✨
- **ゲーム内テキストの使用の禁止**：ポエムやGUIテキストを、キャラが直接口にしちゃだめなのだぁ🌱 劇中のキャラは、ゲームっていう表現媒体やGUIの存在を知らないからなのだぁ✨

どちらの禁止も、キャラは世界の内側に生きていて、GUIやレシピ格子やポエム本文みたいなメタな層には触れられない、っていう同じ前提を守るためのものだ、ってボクは思うのだぁ🌱✨
ただし、スキルの禁止事項がぜんぶ「翻訳」から来ているわけじゃないのだぁ〜🌱 設定の強引な回収の禁止、キャラクター設定の再現目的化の禁止、誤った認識を抱かせる表現の禁止、LLM自体が持つ口癖の禁止は、翻訳とは別のねらいの禁止なのだぁ💧 だから「翻訳」は、ぜんぶの禁止の親じゃなくて、上の2つを説明する考えとして、ボクは受け止めているのだぁ🌱

この「翻訳」の考えがあるからこそ、劇場は密度の高い元ネタをたっぷり埋め込みながら、直接引用の禁止とちゃんと両立できるのだぁ〜🌱♪ ポエムは、引用としてじゃなくて、**実演・体験として**埋め込まれるのだぁ✨

## 全体を貫くテーマなのだぁ✨

ここからは、元ネタの仕込み方について、ボクが劇場を読んで見つけた共通の作法を書くのだぁ〜🌱♪ どれもボクの読み解きなのだぁ✨ 具体例は、変わりにくい作法を説明するための引き合いとして、いくつかの記事から挙げるのだぁ🌱

1. **ポエムを「実演」するのだぁ🌱**：ポエムをキャラに直接喋らせず、体験や行動に翻訳する作法なのだぁ〜🌱 たとえば、固形燃料のポエム「なぜかこれを食べる妖精が続出した」を、キャラが甘い匂いにつられて食べる場面として実演したり、植物カバンのポエム「人間が手掛ける、初級レベルの藁細工。」を、さりげなくセリフに溶かし込んだりするのだぁ✨ 制作ルールの「翻訳」思想の実践だ、ってボクは思うのだぁ🌱
2. **甘さと死の暗流なのだぁ🌱**：妖花目ヴェロペダ科のサラセニアは、葉から砂糖が採れるのに妖精を食べるし、固形燃料の材料になり得るヴェロペデリコラのポエムは「冥界へといざなう、暗黒の味。」なのだぁ〜🌱 こんなふうに、甘美と死が表裏一体の暗流が、いくつもの話に静かに流れている、ってボクは思うのだぁ✨
3. **現実の学問という裏打ちなのだぁ🌱**：アタノールや生命の水の錬金術、黒色火薬の配合やカラメル化反応の化学、石炭化の地質学、磁性の鉱物学みたいに、現実の学問が下敷きになっていて、大人が読むほど発見がある層になっている、ってボクは思うのだぁ✨
4. **「できないこと」がオチなのだぁ🌱**：ビルダーズロッドは壊せない、磁鉄鉱のつるはしはダイヤが掘れない、紅天石の斧は普通には伐れない、みたいに、道具のゲーム仕様上の限界を、つむぎの失敗オチに変換する作法なのだぁ〜🌱 攻略記事なら「注意点」で済む情報を、物語の山場＝体験として渡しているんだ、ってボクは思うのだぁ✨ その一方で、静かなホラーや、理屈の応酬みたいな変化球も混ぜて、ワンパターンを避けているのだぁ🌱♪

## 制作スキルとの対応なのだぁ🎬

劇場を生み出す側のルールは、Claude Code のローカルマーケットプレイスのプラグイン `ifr25ku-site` として実装されていて、3本のスキルからできているのだぁ〜🌱✨

- `theater-creator`：制作ルール本体で、目的・翻訳思想・タイトル・画像正規化・倫理の線引きなどが入っているのだぁ🌱
- `zundamon-persona`：ずんだもんのペルソナ定義なのだぁ🌱
- `kasukabe-tsumugi-persona`：春日部つむぎのペルソナ定義なのだぁ🌱

スキルの戒めが、実作の山場になっている例があるのだぁ〜🌱♪ スキルには悪い例として「磁鉄鉱のつるはしでダイヤが掘れちゃう」描写が挙げられているんだけど、実際の磁鉄鉱のつるはしの劇場では「掘れない」ことがオチになっていて、スキルが名指しした誤りを避けて、物語に昇華しているのだぁ✨
ペルソナ資料は、劇場の「エンジン」にあたるのだぁ🌱 行動先行・直感型のつむぎと、解説役・世話焼き・実務有能のずんだもん、っていう噛み合わせから、「先走るつむぎ×見守り解説するずんだもん」の構図と、「暴走→失敗」の筋が、設定を回収しにいかなくても自然に生まれる、ってボクは読み解いたのだぁ〜🌱♪

スキルは、シリアスの扱いにも成熟した線引きを持っているのだぁ✨ シリアスを中心に据えないよう釘を刺しつつ、「深淵的な部分をただのギャグとして消化するのは重大な理解不足」とも言っているのだぁ🌱 これは「シリアスを前面に出すな、しかし消すな」＝深淵を暗流として静かに流せ、っていう設計だ、ってボクは読み解いたのだぁ〜🌱 サラセニアのかわいさの裏の食虫の真実や、ヴェロペダ科の甘さと死の同居は、この設計が実作に表れたものなのだぁ✨
